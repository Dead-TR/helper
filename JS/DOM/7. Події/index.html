Найчастіші події:

  • onClick -- Клік по елементові
  • onload -- Коли завантажився документ, або частина документа
  • onchange -- Для перемикачів
  • onMouseOver / onMouseOut -- миша в зоні/за її межами
  • onMouseDown / onMouseUp -- ? після відпускання клавіші миші (Натис/Відпустив)
  • onKeyDown / onKeyPress / onKeyUp -- Натис на кнопку/натиснута/відпустив
  • onfocus / onblur -- коли елемент в фокусі/не в фокусі
